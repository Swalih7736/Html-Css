Demonstrating CSS Styling Methods

Inline Styling

This text is styled using an *inline style* attribute directly on the div element. This method should generally be avoided for large projects as it doesn't allow for easy reuse.

External Styling

This text is styled using an *external style sheet* (external_styles.css) linked in the <head> section. This is the *most recommended* method for large-scale projects.

Internal Styling

This text is styled using an *internal style sheet* defined within the <style> tags in the <head> section. This is useful for single-page applications or when a style is unique to one page.
